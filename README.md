## 📍 Overview

This is a simple Wallpaper created in HTML, CSS and JavaScript for Lively Wallpaper, available on **Microsoft Store**.

---

## 👾 Features

<code>❯ Customizable display name</code>

<code>❯ All colors customizable</code>

<code>❯ Glitch effect speed customizable </code>

<code>❯ Toggleable time and date</code>

<code>❯ Custom background and sign brightness</code>

## 🚀 Getting Started

### ☑️ Prerequisites

Before getting started with lively-neon-sign, ensure your runtime environment meets the following requirements:

- **App:** Lively Wallpaper - [_Microsoft Store_](https://www.microsoft.com/store/productId/9NTM2QC6QWS7?ocid=pdpshare)

### 🤖 Usage

1. Download the ZIP file in the **releases** section.
2. Import the **ZIP** in **Add custom wallpaper** on Lively Wallpaper menu.
3. Enjoy the wallpaper
